# Governance

## 1. 维护方式 / How The Project Is Maintained

中文：
说明谁在维护、如何做决策、谁有最终合并权限。

English:
Explain who maintains the project, how decisions are made, and who has final merge authority.

## 2. 决策原则 / Decision Principles

中文：
- 优先用户价值
- 优先长期可维护性
- 尽量小步演进

English:
- Prioritize user value
- Prioritize long-term maintainability
- Prefer incremental evolution

## 3. 变更升级路径 / Escalation Path

中文：
说明普通改动、重大改动和争议改动分别怎么推进。

English:
Explain how routine changes, major changes, and disputed changes should move forward.
