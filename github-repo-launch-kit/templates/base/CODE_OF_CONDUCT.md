# Code of Conduct

## 1. 我们希望的协作方式 / Expected Collaboration

中文：
这个项目欢迎认真、友善、具体的交流。我们希望所有参与者都能以建设性方式沟通，尊重不同背景和经验水平的人。

English:
This project welcomes thoughtful, kind, and specific collaboration. We expect everyone to communicate constructively and respect people with different backgrounds and experience levels.

## 2. 不可接受的行为 / Unacceptable Behavior

中文：
- 骚扰、歧视或人身攻击
- 恶意嘲讽或羞辱
- 泄露他人隐私信息
- 明知会破坏讨论氛围的行为

English:
- Harassment, discrimination, or personal attacks
- Malicious ridicule or humiliation
- Sharing private information without consent
- Behavior that knowingly damages the collaboration environment

## 3. 处理方式 / Enforcement

中文：
维护者可以要求修改、隐藏、删除不当内容，并在必要时限制参与权限。

English:
Maintainers may request changes, hide or remove inappropriate content, and restrict participation when necessary.
