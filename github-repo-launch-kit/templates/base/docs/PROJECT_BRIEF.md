# Project Brief

## 1. 项目目标 / Project Goal

中文：
用 3 到 5 句说明项目为什么存在、核心用户是谁、它解决的最重要问题是什么。

English:
Use 3 to 5 sentences to explain why the project exists, who the core users are, and which problem matters most.

## 2. 非目标 / Non-Goals

中文：
写清楚这个项目不打算解决什么，避免无限扩张。

English:
State clearly what the project is not trying to solve to avoid uncontrolled scope growth.

## 3. 成功标准 / Success Criteria

中文：
定义什么结果代表这个项目真的有价值。

English:
Define what outcomes would mean the project is genuinely valuable.
