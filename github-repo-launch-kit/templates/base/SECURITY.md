# Security Policy

## 1. 报告方式 / How To Report

中文：
如果你发现了安全问题，请不要直接公开提 issue。请通过以下方式私下联系维护者：

`{{SECURITY_CONTACT}}`

English:
If you discover a security issue, please do not open a public issue first. Contact the maintainers privately at:

`{{SECURITY_CONTACT}}`

## 2. 报告内容建议 / What To Include

中文：
- 影响范围
- 复现步骤
- 风险判断
- 受影响版本
- 如果可能，给出修复建议

English:
- Impact scope
- Reproduction steps
- Risk assessment
- Affected versions
- A fix suggestion if possible

## 3. 响应预期 / Response Expectation

中文：
维护者会尽快确认问题、评估严重度，并在合适时机公开修复信息。

English:
Maintainers will acknowledge the report, evaluate severity, and disclose the fix appropriately when ready.
