# Support

## 1. 去哪里提问 / Where To Ask

中文：
- 使用问题和反馈：GitHub Discussions 或 issue
- 明确 bug：bug report issue template
- 安全问题：见 [SECURITY.md](SECURITY.md)

English:
- Usage questions and feedback: GitHub Discussions or an issue
- Clear bugs: use the bug report issue template
- Security issues: see [SECURITY.md](SECURITY.md)

## 2. 提问前建议 / Before You Ask

中文：
- 先看 [README.md](README.md)
- 先跑一次 `node dist/cli.js --help`
- 先看 benchmark 和 workflow 报告

English:
- Start with [README.md](README.md)
- Run `node dist/cli.js --help` once
- Check the command output and current project examples
